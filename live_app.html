ระบบการบันทึกการสอนด้วยระบบ On Demand
- เป็นระบบที่เป็นเว็บแอปพลิเคชั่น (Web Application) มีฟังก์ชั่นการทำงานอยู่บนเว็บผ่านทางโปรแกรมประเภทบราวเซอร์ และสามารถทำงานได้ในระบบเครือข่ายภายใน (Intranet)โดยติดตั้งโปรแกรมระบบทั้งหมดบนเครื่องคอมพิวเตอร์ที่ทำหน้าที่แม่ข่าย และสามารถเชื่อมต่อทำงานในระบบอินเตอร์เน็ตได้
- เป็นระบบที่ทำงานบนระบบปฏิบัติการ Linux Server
- เป็นระบบที่รองรับการทำงานด้วยภาษาเว็บโปรแกรมมิ่ง (Web Programming Language) เช่น PHP, HTML, Java Script เป็นต้น
- เป็นระบบที่รองรับการทำงานร่วมกับเซิร์ฟเวอร์ (Web Server)
- เป็นระบบที่สนับสนุนการทำงานร่วมกับโปรแกรมบราวเซอร์ เช่น Microsoft Internet Explorer
- เป็นระบบที่สามารถบันทึกการสอนด้วยระบบอัตโนมัติ โดยจัดเก็บชนิดของไฟล์ในรูปแบบ mp4
- เป็นระบบที่มีการเรียกใช้งานได้ด้วยระบบการฟิก โดยสามารถดูได้แบบตารางปฏิทิน
- ระบบจะต้องใช้รูปแบบการทำงานแบบ VDO STREAMING
- ระบบสามารถกรอกรายละเอียดของการบันทึกได้

ระบบการส่งสัญญาณ
- เป็นระบบที่สามารถรับข้อมูลสัญญาณภาพที่เป็น HD ได้
- เป็นระบบที่สามารถ STREAM สัญญาณ โดยไม่จำกัดจำนวนผู้รับสัญญาณ
- เป็นระบบที่สามารถกำหนดคุณภาพสัญญาณที่ส่งออกได้

ระบบการรับสัญญาณ
- เป็นระบบที่เป็นเว็บแอปพลิเคชั่น (Web Application) มีฟังก์ชั่นการทำงานอยู่บนเว็บผ่านทางโปรแกรมประเภทบราวเซอร์ และสามารถทำงานได้ในระบบเครือข่ายภายใน (Intranet)โดยติดตั้งโปรแกรมระบบทั้งหมดบนเครื่องคอมพิวเตอร์ที่ทำหน้าที่แม่ข่าย และสามารถเชื่อมต่อทำงานในระบบอินเตอร์เน็ตได้
- เป็นระบบที่รองรับการทำงานด้วยภาษาเว็บโปรแกรมมิ่ง (Web Programming Language) เช่น PHP, HTML, Java Script เป็นต้น
- เป็นระบบที่สนับสนุนการทำงานร่วมกับโปรแกรมบราวเซอร์ Microsoft Internet Explorer, หรือ Google Chromeเป็นต้น
- เป็นระบบที่มีการ Login เพื่อตรวจสอบชื่อผู้ใช้ (Username)และรหัสผ่าน(password) ของผู้ใช้งานในการรับชมการถ่ายทอดสด
- เป็นระบบที่สามารถรับชมการถ่ายทอดสดได้ เป็นระบบที่สามารถใช้งานได้บนเครื่องคอมพิวเตอร์ TABLET และระบบปฏิบัติการ WINDOWS,IOS,ANDROID ได้
